# 涩批通识
在人类几千年的历史长河中，没有一种人会比`LSP`更让人感到惊讶。LSP本意是一种调侃。但随着社会的不断发展，LSP们逐渐的成为了一个强有力的社会群体，他们有着独立的文化与受众范围，游走于网络与现实之中。他们是人类之美的追求者，在文明中璀璨发光。作为LSP，也会面临着很多困扰，比如传输艺术鉴赏用资源时常被判定为黄色内容，面对群友们的伦理责问时常常不知如何应答……可见，LSP也需要学习很多知识来保护自己。本书将分为8个章节，分别从多个角度与学科简述作为LSP需要学习的内容。

在此感谢各位网友的支持，没有你们的努力，这本书不会面世。
